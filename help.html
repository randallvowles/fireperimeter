Using the Webpage

To filter the results either enter the range for each parameter, enter the min field to create a greater-than parameter (leave the max blank), or create a less-than parameter by entering the max (leave the min blank).

Clicking on the Station ID will take you to the MesoWest page for that station, where you can access graphical displays, timeseries data from all sensors, and additional metadata.

Cells highlighted yellow indicate a QC flag associated with that observation, hover over the cell for the QC flag description.
